CSS 规范学习

基于 W3C 官方规范，理解 CSS 原理

学习路线

CSS 规则繁多的根本原因是它们背后有统一的底层模型。本项目的目标是通过阅读规范、理解原理、编写示例来系统掌握 CSS。

  • 03 Selectors 选择器 - 如何选中元素
  • 04 Values & Units 值和单位 - px, em, rem, %, vw...

阶段二：布局系统

  • 08 Grid 网格布局
